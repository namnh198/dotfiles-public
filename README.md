## Neovim setup

### Requirements
- [Neovim](https://neovim.io) >= **0.9.0**
- [Git](https://git-scm.com) >= **2.19.0**
- [Lazyvim](https://www.lazyvim.org)
- [NerdFont](https://github.com/ryanoasis/nerd-fonts)
- [lazygit](https://github.com/jesseduffield/lazygit) (Optional)
- C compiler (use for `nvim-treesitter`) See [here](https://github.com/nvim-treesitter/nvim-treesitter#requirements)
- [ripgrep](https://github.com/BurntSushi/ripgrep) & [fd](https://github.com/sharkdp/fd) (use for `telescope`)
- Terminal support true color and undercurl:
  - [kitty](https://github.com/kovidgoyal/kitty)
  - [alacritty](https://github.com/alacritty/alacritty)
  - [iterm2](https://iterm2.com/)

## Fish shell
- [Fish shell](https://fishshell.com/)
- [Fisher](https://github.com/jorgebucaran/fisher) - Plugin Manager
- [Tide](https://github.com/IlanCosman/tide) - Shell theme. `fisher install IlanCosman/tide@v6`
- [NerdFont](https://github.com/ryanoasis/nerd-fonts)
- [z](https://github.com/jethrokuan/z) - Directory jumpping
- [eza](https://github.com/eza-community/eza) - `ls` replacement 
- [ghq](https://github.com/x-motemen/ghq) - Local Git repository organizer
- [fzf](https://github.com/PatrickF1/fzf.fish) - Interactive finder
